## Getting Started

Clone the repo:
```bash
git clone git@github.com:longislandjavascript/www.lijs.org.git
```

Install the dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
